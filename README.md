# 🍷 Wine Quality Prediction App

## 📌 Project Overview
The **Wine Quality Prediction App** is a Machine Learning application that predicts the quality of wine based on its chemical properties.  

This project uses a trained machine learning model to analyze different wine characteristics and classify the wine quality. The goal of this project is to demonstrate how machine learning can be used to solve real-world prediction problems.

---

## 🚀 Features
- Predict wine quality using machine learning
- Simple and interactive user interface
- Real-time predictions
- User inputs different chemical properties of wine
- Displays predicted wine quality instantly

---

## 🧠 Machine Learning Model

The model is trained using the **Wine Quality Dataset**.

### Input Features

The prediction is based on the following features:

- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol

### Output

- **Predicted Wine Quality Score**
